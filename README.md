# 🛡️ Provost Inc — Azure Zero Trust SOC Lab

## Project Overview
A fully documented, enterprise-grade cybersecurity lab simulating a 
SOC Tier 2/3 environment for Provost Inc, a fictional regulated 
financial services company.

## Certifications Demonstrated
- CompTIA Security+
- AZ-104 Azure Administrator
- AZ-500 Azure Security Engineer
- SC-200 Microsoft Security Operations Analyst
- SC-300 Microsoft Identity & Access Administrator
- ISO 27001 Information Security Management
- ISO 42001 AI Management System

## Lab Environment
- Cloud: Microsoft Azure (Free account / Pay-As-You-Go)
- SIEM: Microsoft Sentinel
- Identity: Microsoft Entra ID
- Security: Microsoft Defender for Cloud
- Automation: Azure Logic Apps
