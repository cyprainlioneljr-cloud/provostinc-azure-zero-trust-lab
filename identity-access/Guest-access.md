# Provost Inc External Identity (B2B) Access

## Objective
Simulate secure collaboration with an external contractor/partner using 
Entra ID B2B guest access.

## Configuration
| Setting | Value |
|---------|-------|
| Guest type | B2B invited external user |
| Identity marker | #EXT# in user principal name |
| Onboarding | Invitation email + acceptance flow |

## Why This Matters
Regulated enterprises routinely work with external auditors, contractors, 
and partners. B2B guest access allows controlled collaboration without 
creating fully internal accounts, while still applying Conditional Access 
and monitoring to those guests.
